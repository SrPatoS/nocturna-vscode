# Nocturna Theme for VS Code

This is a port of the [Nocturna IntelliJ Theme](https://plugins.jetbrains.com/plugin/26078-nocturna) to Visual Studio Code.

## Installation

### Method 1: Extension Folder
1. Copy the `nocturna-vscode` folder to your VS Code extensions directory:
   - Windows: `%USERPROFILE%\.vscode\extensions`
   - macOS/Linux: `~/.vscode/extensions`
2. Restart VS Code.
3. Open Color Theme selection (`Ctrl+K Ctrl+T`) and select **Nocturna**.

### Method 2: Local Install with vsce
If you have `vsce` installed:
1. Open terminal in `nocturna-vscode` directory.
2. Run `vsce package`.
3. Install the generated `.vsix` file in VS Code.
